Lenovo ThinkCentre M910q Tiny (Desktop)

Previous Next

Description

This desktop is smaller than a laptop and can handle any work you throw at it. With the 3.80 GHz Intel Core i7-7700T processor, 8 GB RAM, and a 512 GB SSD using the new NVMe interface (up to 4X as fast as existing SATA SSDs), you don't have to spend a lot to get workstation-level performance. The onboard video provides dual monitor output at 2K each.

Highlights

  • Smallest desktop footprint yet, can easily be hidden behind a monitor in any workstation scenario
  • CPU is new Intel Core i7-7700T 3.80 GHz from the Kaby Lake processor family
  • 8 GB RAM
  • 512 GB SSD with new NVMe interface, up to 4X as fast as existing SATA SSDs
  • Support for dual monitor output at 2K resolution each, via two DisplayPort ports

Specifications

  • Processor: Intel Core i7-7700T 3.80 GHz (4 cores, 8 with HT)
  • RAM: 8 GB
  • Storage: 512 GB SSD with NVMe interface
  • Video output: Dual DisplayPort, up to 2K each
  • Warranty: 4 Year Onsite Warranty
  • Weight: 2.91 lb (1.32 kg)
  • Size (WxDxH): 7.05" X 7.20" X 1.36"; 179mm X 182.9mm X 34.5mm
